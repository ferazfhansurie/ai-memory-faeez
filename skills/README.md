# Storytelling and Content Skills

Portable copies of the pure storytelling and content-strategy playbooks from Faeez's Claude workspace. These focus on ideas, narrative, planning, pacing, and execution — not AI image or video generation.

## Storytelling

- `storytelling/ad-story.md` — ad story spines, hooks, stakes, transformation, and CTA design.
- `storytelling/unique-ad.md` — stop-the-scroll structures, emotional layering, and anti-cliche filters.
- `storytelling/fathopes-week.md` — FatHopes Energy's weekly superhero mini-series and organic content system.

## Content production

- `content-creation/reel-tempo.md` — short-form reel pacing, beat structure, cuts, and safe zones.

These are intentionally stored as plain Markdown so any AI assistant can read them without Claude-specific runtime dependencies. The source files were selected from the project Claude configuration at `.claude/` and `aios/.claude/`.
