# Storytelling and Content Skills

Portable copies of the storytelling and content-production playbooks from Faeez's Claude workspace.

## Storytelling

- `storytelling/ad-story.md` — ad story spines, hooks, stakes, transformation, and CTA design.
- `storytelling/unique-ad.md` — stop-the-scroll structures, emotional layering, and anti-cliche filters.
- `storytelling/fathopes-week.md` — FatHopes Energy's weekly superhero mini-series storytelling system.

## Content production

- `content-creation/motionboards-content-factory/SKILL.md` — the five-stage UGC campaign pipeline.
- `content-creation/reel-tempo.md` — short-form reel pacing, beat structure, cuts, and safe zones.
- `content-creation/seedance-prompt.md` — Seedance motion prompting, camera direction, references, and prompt compression.
- `content-creation/nano-banana.md` — Nano Banana image prompting, composition, references, and iteration.

These are intentionally stored as plain Markdown so any AI assistant can read them without Claude-specific runtime dependencies. The source files were selected from the project Claude configuration at `.claude/` and `aios/.claude/`; the newer `aios/.claude` variants are used where the two copies differ.
